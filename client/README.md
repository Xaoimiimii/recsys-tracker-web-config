# Recsys Tracker Web Config

A web application for configuring and managing tracking for recommendation systems.

## Requirements

- Node.js (version 16 or higher)
- npm or yarn

## Installation

```bash
npm install
```

## Running the Application

### Development mode
```bash
npm run dev
```

### Build for production
```bash
npm run build
```

### Preview production build
```bash
npm run preview
```

## Project Structure

```
├── app/              # Main pages
├── components/       # React components
├── lib/             # Utilities and constants
├── styles/          # CSS files
├── types/           # TypeScript type definitions
└── public/          # Static files
```

## Tech Stack

- React 19
- TypeScript
- Vite
- React Router
- Motion (animations)
