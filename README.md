# Recsys Tracker Web Config

A web application for configuring and managing tracking for recommendation systems.

## Project Structure

```
├── client/           # Frontend application
└── README.md        # This file
```

## Getting Started

### Client Application

Navigate to the `client` directory and follow the instructions in the [client README](./client/README.md).

```bash
cd client
npm install
npm run dev
```

## Tech Stack

- React 19
- TypeScript
- Vite
- React Router
